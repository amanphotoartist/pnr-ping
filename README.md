# 🚆 PNRPing

PNRPing automatically tracks your train PNR and sends email updates whenever your ticket status changes.

## Features

- 🚆 Track train PNR
- 📧 Email notifications
- ⏰ Automatic status monitoring
- ✅ Auto stop after confirmation

## Tech Stack

### Frontend
- React 19
- Vite 7
- Tailwind CSS 4

### Backend
- FastAPI
- SQLAlchemy
- APScheduler

### Database
- PostgreSQL (Supabase)

### Notifications
- Resend Email

### PNR Data
- RapidAPI
